# 724events - Debug & Tests 🛠️

Débogage et finalisation d’un site one-page développé pour une agence événementielle afin de corriger les dysfonctionnements, stabiliser les fonctionnalités et garantir une expérience utilisateur fluide.

Le projet consistait à analyser le code existant, identifier les bugs présents sur le site puis effectuer les corrections nécessaires à l’aide des outils de développement React et JavaScript.

## Features

- Correction de bugs JavaScript et React
- Débogage des composants interactifs
- Analyse du site avec Chrome DevTools
- Utilisation de React Developer Tools
- Vérification du responsive design
- Validation des scénarios utilisateurs
- Mise en place et finalisation d’un cahier de recette
- Réalisation de tests fonctionnels et unitaires
- Stabilisation des fonctionnalités du site
- Optimisation de l’expérience utilisateur

## Tech Stack
